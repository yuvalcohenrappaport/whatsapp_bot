  <action>
Create `scripts/test-voice.ts` — a standalone script that tests both voice modules against the live ElevenLabs API. Run with `npx tsx scripts/test-voice.ts`.

**Structure:**

1. **Imports:**
   - `import { transcribe } from '../src/voice/transcriber.js'`
   - `import { textToSpeech } from '../src/voice/tts.js'`
   - `import { writeFileSync } from 'fs'`
   - `import pino from 'pino'`

2. **Logger:** Create `const logger = pino({ level: 'debug' })` for verbose output during testing.

3. **OGG magic bytes helper:**
   ```typescript
   function isOggContainer(buffer: Buffer): boolean {
     return buffer.length >= 4 &&
       buffer[0] === 0x4F && buffer[1] === 0x67 &&
       buffer[2] === 0x67 && buffer[3] === 0x53;
   }
   ```

4. **TTS Test (run first — generates audio we can use for STT test):**
   - Define 5 Hebrew test sentences:
     ```
     'שלום, מה שלומך?'
     'אני יכול לעזור לך עם זה.'
     'הפגישה נדחתה לשבוע הבא.'
     'תודה רבה על עזרתך.'
     'הדוח המחקרי הוגש בזמן.'
     ```
   - Loop over sentences. For each:
     - Call `const audio = await textToSpeech(sentence, logger)`
     - Assert OGG magic bytes: `if (!isOggContainer(audio)) throw new Error('TTS output is not OGG')`
     - Log: `console.log(`TTS "${sentence.slice(0, 25)}..." => ${audio.length} bytes, OGG: true`)`
   - Save the first sentence's audio to `./test-output-tts.ogg` for manual playback: `writeFileSync('./test-output-tts.ogg', firstAudio)`
   - Log: `console.log('Saved test-output-tts.ogg for manual playback')`

5. **STT Test (use the TTS output as input for a round-trip test):**
   - Take the first TTS output buffer (already OGG/Opus)
   - Call `const transcript = await transcribe(firstAudio, logger)`
   - Log: `console.log('STT round-trip transcript:', transcript)`
   - This tests the full round-trip: text -> TTS -> OGG -> STT -> text
   - Do NOT assert exact text match (TTS -> STT will have differences) — just verify it returns a non-empty Hebrew string
   - Assert: `if (!transcript || transcript.length === 0) throw new Error('STT returned empty transcript')`

6. **Summary:**
   - Print a final summary:
     ```
     console.log('\n--- Results ---')
     console.log(`TTS: ${successCount}/5 sentences produced valid OGG`)
     console.log(`STT: Round-trip transcript: "${transcript}"`)
     console.log('Listen to test-output-tts.ogg to verify Hebrew naturalness')
     ```

7. **Error handling:**
   - Wrap the entire script in an async IIFE or top-level try/catch
   - On error, log the error and `process.exit(1)`

**Important:**
- Do NOT import anything from messageHandler, index.ts, or any pipeline code
- The script only touches the two voice modules and the ElevenLabs API
- The `test-output-tts.ogg` file is written to project root for easy access (add to .gitignore if not already covered)
- This script makes REAL API calls that cost money — the checkpoint task below will confirm user wants to run it
