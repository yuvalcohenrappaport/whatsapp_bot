```typescript
function buildSuggestionText(title: string, date: Date, location?: string | null): string {
  const dateStr = date.toLocaleString('he-IL', {
    month: 'long',
    day: 'numeric',
    hour: '2-digit',
    minute: '2-digit',
    timeZone: 'Asia/Jerusalem',
  });
  const locationPart = location ? `, ${location}` : '';
  return `📅 להוסיף '${title}' ב-${dateStr}${locationPart}? השב ✅ או ❌`;
}
```

**Helper: `isDuplicate`** — check if a pending suggestion with same groupJid + title + eventDate (within 1-hour window) already exists:
```typescript
function isDuplicate(groupJid: string, title: string, eventDate: number): boolean {
  for (const s of pendingSuggestions.values()) {
    if (
      s.groupJid === groupJid &&
      s.title === title &&
      Math.abs(s.eventDate - eventDate) < 3_600_000
    ) {
      return true;
    }
  }
  return false;
}
```

**Helper: `startTtlTimer`** — creates a setTimeout that silently removes the suggestion from Map and DB on expiry:
```typescript
function startTtlTimer(id: string, remainingMs: number): NodeJS.Timeout {
  return setTimeout(() => {
    pendingSuggestions.delete(id);
    deletePendingSuggestion(id);
    logger.debug({ id }, 'Suggestion TTL expired — silently discarded');
  }, remainingMs);
}
```

**Export: `createSuggestion`** — called from the pipeline for each extracted date. Sends suggestion message to group and stores in Map + DB.

Parameters: `(groupJid: string, extracted: { title: string; date: Date; location?: string; description?: string; url?: string }, calendarId: string, calendarLink: string, sourceMessageId: string, senderName: string | null): Promise<void>`

Logic:
1. Check `isDuplicate(groupJid, extracted.title, extracted.date.getTime())`. If true, log debug and return.
2. Get `sock` from `getState()`. If null, log warn and return.
3. Build suggestion text with `buildSuggestionText(extracted.title, extracted.date, extracted.location)`.
4. Send message: `const sent = await sock.sendMessage(groupJid, { text: suggestionText })`.
5. Get `suggestionMsgId = sent?.key?.id`. If null, log warn and return.
6. Create suggestion ID with `crypto.randomUUID()`.
7. Calculate `expiresAt = Date.now() + SUGGESTION_TTL_MS`.
8. Start TTL timer: `const timer = startTtlTimer(id, SUGGESTION_TTL_MS)`.
9. Build description string: `const desc = extracted.description ?? buildDefaultDescription(extracted, senderName, groupJid)` where `buildDefaultDescription` creates something like `"${msg.body}\n\nSent by: ${senderName}\nGroup: ${groupJid}"` — actually, pass the original message body via the `description` field or build from title + sender.
10. Store in Map: `pendingSuggestions.set(id, { ...all fields, timer })`.
11. Persist to DB: `insertPendingSuggestion({ id, groupJid, suggestionMsgId, title: extracted.title, eventDate: extracted.date.getTime(), location: extracted.location ?? null, description: extracted.description ?? null, url: extracted.url ?? null, calendarId, calendarLink, sourceMessageId, senderName, expiresAt, createdAt: Date.now() })`.
12. Log info: `{ id, title: extracted.title, groupJid }`.

**Export: `handleConfirmReject`** — called from the pipeline callback to check if a quoted-reply is a ✅/❌ response to a pending suggestion.

Signature: `async function handleConfirmReject(groupJid: string, msg: { id: string; senderJid: string; senderName: string | null; body: string; timestamp: number; fromMe?: boolean }, quotedMessageId: string | null): Promise<boolean>`

Logic:
1. If `!quotedMessageId`, return false.
2. Look up: first check in-memory Map by iterating values for `s.suggestionMsgId === quotedMessageId`. If not found, try DB: `getPendingSuggestionByMsgId(quotedMessageId)`. If neither, return false.
3. Check body: `const trimmed = msg.body.trim()`. If `trimmed !== '✅' && trimmed !== '❌'`, return false.
4. If `trimmed === '❌'`: Reject — clear timer, delete from Map, delete from DB. Log debug. Return true.
5. If `trimmed === '✅'`: Confirm — call `confirmSuggestion(groupJid, suggestion)`. Return true.

**Private: `buildEventDescription`** — builds the event description from suggestion fields:
```typescript
function buildEventDescription(s: { title: string; description: string | null; url: string | null; senderName: string | null; sourceMessageId: string }): string {
  const parts: string[] = [];
  if (s.description) parts.push(s.description);
  if (s.url) parts.push(`Link: ${s.url}`);
  if (s.senderName) parts.push(`Suggested by: ${s.senderName}`);
  return parts.length > 0 ? parts.join('\n') : s.title;
}
```
