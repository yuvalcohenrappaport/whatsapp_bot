**Expected:** Bot sends a Hebrew message opening with "ראיתי שבחרתם רומא! הנה כמה רעיונות:" followed by 3-4 activity tips as bullet points.

**Expected:** No second proactive message is sent; server logs show `canSendProactive` returned false.
**Why human:** In-memory state requires runtime observation across two processTripContext calls.

**3. Weekly digest with open questions**

**Test:** Ensure a group has unresolved `open_question` records in the DB (less than 30 days old). Trigger `generateWeeklyDigest` for that group (manually or via cron).
**Expected:** Digest ends with "🧳 שאלות פתוחות לטיול:" section listing each question with Hebrew age indicator (e.g., "❓ האם המקום כשר? (לפני 3 ימים)").
**Why human:** Requires live DB state and either manual digest invocation or waiting for the weekly cron.
