  <how-to-do-it>
    **Step 1: ElevenLabs account and API key**
    1. Go to elevenlabs.io and sign up (or log in if you have an account)
    2. Navigate to Profile -> API Keys
    3. Create a new API key and copy it

    **Step 2: Record audio samples**
    - Use Voice Memos on iPhone (or any app that saves clean audio)
    - Record 3-5 clips of yourself speaking naturally in Hebrew
    - Each clip: 30-120 seconds, clear speech, no background noise
    - Quality bar: clips that would let someone recognise your voice
    - Save as MP3 if possible; WAV or M4A also works (avoid WhatsApp OGG — too compressed)

    **Step 3: Create Instant Voice Clone**
    1. In ElevenLabs: go to Voices -> Add Voice -> Instant Voice Cloning
    2. Upload your 3-5 audio clips
    3. Name the voice (e.g. "Yuval Hebrew")
    4. Click Create
    5. Once created, open the voice and copy its Voice ID (shown in the URL or in Voice Settings)

    **Step 4: Test the voice clone**
    1. In ElevenLabs Voices, select your cloned voice
    2. In the text box, type 5 test sentences in Hebrew (e.g. your typical WhatsApp replies)
    3. Generate speech using model: `eleven_multilingual_v2`
    4. Listen and verify: does it sound recognisably like you? (quality bar: "people who know me would say that sounds like me")

    **Step 5: Add real credentials to .env**
    Open `/home/yuval/whatsapp-bot/.env` and replace the placeholder lines:
    ```
    ELEVENLABS_API_KEY=your_real_api_key_here
    ELEVENLABS_DEFAULT_VOICE_ID=your_real_voice_id_here
    ```
  </how-to-do-it>
