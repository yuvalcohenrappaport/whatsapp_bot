**Interface:**
```ts
export interface BriefingInput {
  groupJid: string;
  destination: string;
  calendarId: string | null;
  destTz: string;
  todayIso: string; // 'YYYY-MM-DD' in destTz
  coords: { lat: number; lon: number } | null; // from metadata.coords; null = resolve on first call
  openWeatherApiKey: string | null; // from config.OPENWEATHER_API_KEY
}

export async function runDayOfBriefing(input: BriefingInput): Promise<void>
```

**Enrichment gathering** (each step in try/catch; failure = log warn + set that field to null/[]):

1. **Calendar events**: call `listUpcomingEvents(calendarId ?? 'primary', 1)` — but the existing function takes `(calendarId, daysAhead)` and returns events up to `daysAhead` days ahead. Filter returned events to `todayIso` (compare `event.date.startsWith(todayIso)` where `event.date` is the ISO string from `listUpcomingEvents`). If `calendarId` is null, skip calendar enrichment (return `[]`).

2. **Weather**: if `coords` is not null, call `getDestinationForecast(coords, apiKey, todayIso)`. If `coords` is null AND `destination` AND `apiKey` are set, call `resolveCoords(destination, apiKey)` first, then cache result by calling `upsertTripContext(groupJid, { metadata: JSON.stringify({ ...existingMeta, coords: newCoords }) })`. If `apiKey` is null, skip weather (return `null`).

3. **Transit alerts**: call `transitAlerts(destination, todayIso)`. Returns string|null — null means unknown, not failure.

4. **Open questions**: call `getUnresolvedOpenItems(groupJid)` — synchronous, returns array.

5. **Today's conflicts**: call `getDecisionsByGroup(groupJid)`, filter where:
   - `JSON.parse(row.conflictsWith ?? '[]').length > 0`
   - `row.metadata` parses to an object where `event_time` is a date string that starts with `todayIso` in destTz (convert `event_time` to local date using `new Date(event_time).toLocaleDateString('sv-SE', { timeZone: destTz })` — sv-SE locale produces `YYYY-MM-DD`)

6. **Budget burn**: call `getBudgetRollup(groupJid)`. Format per category: `"${cat}: ${pct}% of ${target} used"` where `pct = Math.round(spent / target * 100)`. Skip categories where `target === 0`.

**Fallback condition**: if `calendarEvents` failed (threw), mark `useFallback = true`. Other source failures do NOT trigger fallback alone — they are omitted from the briefing but enriched briefing still attempted. Only unrecoverable calendar failure AND Gemini composition failure both together trigger the no-events fallback.

Actually per spec: "If ANY enrichment source fails → fallback". Implement this more conservatively: set `useFallback = true` if the Gemini composition call itself throws OR if calendar events threw (calendar is the minimum for the fallback template to work). Other enrichment failures are logged + omitted.

**Fallback template** (locked):
```ts
function buildFallbackMessage(events: Array<{ title: string; date: string }>, destTz: string): string {
  if (events.length === 0) return '🌅 בוקר טוב! אין אירועים ביומן להיום.';
  const lines = events.map(e => `• ${formatTime(e.date, destTz)} — ${e.title}`);
  return `🌅 בוקר טוב! היום ביומן:\n${lines.join('\n')}`;
}
```

`formatTime(isoDate, tz)`: extract HH:MM from the ISO date string in the given timezone using `Intl.DateTimeFormat`. For all-day events (date-only strings like `2026-05-10`), return `'כל היום'`.

**Posting**: `const { sock } = getState(); if (!sock) { logger.warn(...); return; } await sock.sendMessage(groupJid, { text: message });`

**Full error boundary**: wrap the entire function body in try/catch; on uncaught error, log + attempt fallback post; if fallback post also fails, log error + return (never re-throw).

describe('runDayOfBriefing', () => {
  it('happy path: posts Gemini-composed message to group', async () => {
    vi.mocked(cal.listUpcomingEvents).mockResolvedValue([
      { title: 'Vatican Tour', date: '2026-05-10T09:00:00+02:00' },
    ]);
    vi.mocked(ow.getDestinationForecast).mockResolvedValue([
      { dt: 1746860400, temp: 22, description: 'sunny', icon: '01d' },
    ]);
    vi.mocked(gs.transitAlerts).mockResolvedValue('normal');
    vi.mocked(ai.generateText).mockResolvedValue('🌅 בוקר טוב! היום יש סיור בוותיקן בשעה 09:00...');

    await runDayOfBriefing(INPUT);

    const sockMock = vi.mocked(state.getState)().sock!;
    expect(vi.mocked(sockMock.sendMessage)).toHaveBeenCalledWith(
      GROUP,
      expect.objectContaining({ text: expect.stringContaining('בוקר טוב') }),
    );
    expect(vi.mocked(ai.generateText)).toHaveBeenCalledTimes(1);
  });

  it('fallback: calendar throws → posts fallback template, does not call Gemini composition', async () => {
    vi.mocked(cal.listUpcomingEvents).mockRejectedValue(new Error('Calendar 500'));
    vi.mocked(gs.transitAlerts).mockResolvedValue('normal');

    await runDayOfBriefing(INPUT);

    const sockMock = vi.mocked(state.getState)().sock!;
    const sentText = vi.mocked(sockMock.sendMessage).mock.calls[0][1] as { text: string };
    expect(sentText.text).toContain('🌅 בוקר טוב! אין אירועים ביומן להיום.');
    // Gemini composition should NOT be called on calendar failure
    expect(vi.mocked(ai.generateText)).not.toHaveBeenCalled();
  });

  it('fallback: calendar succeeds but Gemini composition throws → posts fallback with event list', async () => {
    vi.mocked(cal.listUpcomingEvents).mockResolvedValue([
      { title: 'Colosseum', date: '2026-05-10T14:00:00+02:00' },
    ]);
    vi.mocked(gs.transitAlerts).mockResolvedValue('normal');
    vi.mocked(ai.generateText).mockRejectedValue(new Error('Gemini timeout'));

    await runDayOfBriefing(INPUT);

    const sockMock = vi.mocked(state.getState)().sock!;
    const sentText = vi.mocked(sockMock.sendMessage).mock.calls[0][1] as { text: string };
    expect(sentText.text).toContain('🌅 בוקר טוב! היום ביומן:');
    expect(sentText.text).toContain('• 14:00 — Colosseum');
  });

  it('fallback: OpenWeather throws → enriched briefing still attempted (not fallback)', async () => {
    vi.mocked(cal.listUpcomingEvents).mockResolvedValue([{ title: 'Dinner', date: '2026-05-10T19:00:00+02:00' }]);
    vi.mocked(ow.getDestinationForecast).mockRejectedValue(new Error('429'));
    vi.mocked(gs.transitAlerts).mockResolvedValue('normal');
    vi.mocked(ai.generateText).mockResolvedValue('🌅 בוקר טוב! הכל בסדר');

    await runDayOfBriefing(INPUT);

    // Gemini composition was still called (weather failure alone doesn't trigger fallback)
    expect(vi.mocked(ai.generateText)).toHaveBeenCalledTimes(1);
  });

  it('no-op when sock is null', async () => {
    vi.mocked(state.getState).mockReturnValue({ sock: null } as any);
    vi.mocked(cal.listUpcomingEvents).mockResolvedValue([]);

    await expect(runDayOfBriefing(INPUT)).resolves.not.toThrow();
  });
});
