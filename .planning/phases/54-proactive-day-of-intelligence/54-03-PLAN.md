**Interface:**
```ts
export interface BriefingInput {
  groupJid: string;
  destination: string;
  calendarId: string | null;
  destTz: string;
  todayIso: string; // 'YYYY-MM-DD' in destTz
  coords: { lat: number; lon: number } | null; // from metadata.coords; null = resolve on first call
  openWeatherApiKey: string | null; // from config.OPENWEATHER_API_KEY
}

export async function runDayOfBriefing(input: BriefingInput): Promise<void>
```

**Fallback trigger rule (LOCKED from 54-CONTEXT.md):** Set `useFallback = true` if ANY of the following throws or returns an error sentinel: calendar events, OpenWeather (resolveCoords or getDestinationForecast), geminiGroundedSearch transitAlerts, getUnresolvedOpenItems, conflicts query (getDecisionsByGroup), getBudgetRollup, or final Gemini composition. No partial enriched briefings are ever posted. Implement by wrapping ALL enrichment steps collectively, not per-step try/catch — a single outer try/catch that sets `useFallback = true` on any throw from the enrichment block.

**Structure:**
```ts
let calendarEvents: Array<{ title: string; date: string }> = [];
let useFallback = false;

// ─── Calendar (always fetched, needed for fallback template too) ──────────────
try {
  if (calendarId) {
    const all = await listUpcomingEvents(calendarId, 1);
    calendarEvents = all.filter(e => e.date.startsWith(todayIso));
  }
} catch {
  useFallback = true;
}

// ─── All other enrichment sources ────────────────────────────────────────────
let weatherSummary: string | null = null;
let transitAlert: string | null = null;
let openQuestions: Array<{ value: string }> = [];
let todayConflicts: Array<{ value: string }> = [];
let budgetLines: string[] = [];
let composedMessage: string | null = null;

if (!useFallback) {
  try {
    // Weather
    if (coords && openWeatherApiKey) {
      const slots = await getDestinationForecast(coords, openWeatherApiKey, todayIso);
      weatherSummary = slots.map(s => `${s.description} ${Math.round(s.temp)}°C`).join(', ') || null;
    } else if (destination && openWeatherApiKey) {
      const resolved = await resolveCoords(destination, openWeatherApiKey);
      if (resolved) {
        const existingMeta = JSON.parse(getTripContext(groupJid)?.metadata ?? '{}');
        upsertTripContext(groupJid, { metadata: JSON.stringify({ ...existingMeta, coords: resolved }) });
        const slots = await getDestinationForecast(resolved, openWeatherApiKey, todayIso);
        weatherSummary = slots.map(s => `${s.description} ${Math.round(s.temp)}°C`).join(', ') || null;
      }
    }

    // Transit alerts
    transitAlert = await transitAlerts(destination, todayIso);

    // Open questions
    openQuestions = getUnresolvedOpenItems(groupJid);

    // Today's conflicts
    const allDecisions = getDecisionsByGroup(groupJid);
    todayConflicts = allDecisions.filter(row => {
      if (!JSON.parse(row.conflictsWith ?? '[]').length) return false;
      const meta = JSON.parse(row.metadata ?? '{}');
      if (!meta.event_time) return false;
      const localDate = new Date(meta.event_time).toLocaleDateString('sv-SE', { timeZone: destTz });
      return localDate === todayIso;
    });

    // Budget burn
    const rollup = getBudgetRollup(groupJid);
    budgetLines = Object.entries(rollup.targets)
      .filter(([, target]) => (target as number) > 0)
      .map(([cat, target]) => {
        const spent = (rollup.spent as Record<string, number>)[cat] ?? 0;
        const pct = Math.round(spent / (target as number) * 100);
        return `${cat}: ${pct}% of ${target} used`;
      });

    // Gemini composition
    const systemPrompt = 'You are a travel assistant composing a morning briefing in Hebrew for a WhatsApp group. Respond with a single natural WhatsApp message (no markdown headers, use emoji sparingly). Output only the briefing text, nothing else.';
    const userContent = `תאריך: ${todayIso}
יעד: ${destination}

אירועים ביומן היום:
${calendarEvents.map(e => `• ${formatTime(e.date, destTz)} — ${e.title}`).join('\n') || 'אין אירועים'}

מזג אוויר: ${weatherSummary || 'לא זמין'}

עדכוני תחבורה: ${transitAlert || 'לא ידוע'}

שאלות פתוחות:
${openQuestions.map(q => `• ${q.value}`).join('\n') || 'אין'}

קונפליקטים להיום:
${todayConflicts.map(c => `• ${c.value}`).join('\n') || 'אין'}

תקציב:
${budgetLines.join('\n') || 'לא הוגדר'}

כתוב בריפינג בוקר עברי, ידידותי, בסגנון WhatsApp. לא יותר מ-300 מילים.`;

    composedMessage = await generateText({ systemPrompt, messages: [{ role: 'user', content: userContent }] });
  } catch {
    useFallback = true;
  }
}
```

**Fallback template** (locked):
```ts
function buildFallbackMessage(events: Array<{ title: string; date: string }>, destTz: string): string {
  if (events.length === 0) return '🌅 בוקר טוב! אין אירועים ביומן להיום.';
  const lines = events.map(e => `• ${formatTime(e.date, destTz)} — ${e.title}`);
  return `🌅 בוקר טוב! היום ביומן:\n${lines.join('\n')}`;
}
```

`formatTime(isoDate, tz)`: extract HH:MM from the ISO date string in the given timezone using `Intl.DateTimeFormat`. For all-day events (date-only strings like `2026-05-10`), return `'כל היום'`.

**Posting**: `const { sock } = getState(); if (!sock) { logger.warn(...); return; } const message = useFallback ? buildFallbackMessage(calendarEvents, destTz) : composedMessage!; await sock.sendMessage(groupJid, { text: message });`

**Full error boundary**: wrap the entire function body in try/catch; on uncaught error, log + attempt fallback post; if fallback post also fails, log error + return (never re-throw).
