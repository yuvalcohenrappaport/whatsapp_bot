### Add to `src/api/routes/googleTasks.ts`

Four new JWT-gated mutation routes. All require `?listId=<listId>` as a query param because the Google Tasks API is list-scoped.

**PATCH /api/google-tasks/items/:taskId/reschedule**
- Body: `{ dueMs: number }` (unix ms)
- Validates body: `typeof body.dueMs === 'number'` → 400 if not.
- Validates `?listId` present → 400 if missing.
- Calls `rescheduleTodoTask(listId, taskId, body.dueMs)`.
- Returns 200 `{ ok: true }` on success, 502 `{ error: 'gtasks_upstream_error' }` on failure.

**PATCH /api/google-tasks/items/:taskId/edit**
- Body: `{ title: string }` (non-empty, max 1024 chars)
- Validates body, validates `?listId`.
- Check if the item has a matching live actionable via `getActionableByTodoTaskId(taskId)` from `src/db/queries/actionables.ts` (this helper is added in Plan 46-01 Task 2 alongside `getApprovedActionableTodoTaskIds`). If found AND `actionable.status === 'approved'`, route through Plan 45-02: call `POST /api/actionables/:id/edit` internally (use `approveActionable` primitive pattern — NOT an HTTP self-call; call `updateActionableTask(actionable.id, body.title)` then `approveActionable(sock, refreshed)` like the /edit route in actionables.ts). If no live actionable, call `editTodoTaskTitle(listId, taskId, body.title)`.
- Returns 200 `{ ok: true }`.

**DELETE /api/google-tasks/items/:taskId**
- Query: `?listId=<listId>`
- Calls `deleteTodoTask(taskId, listId)` (already exists in todoService.ts — signature is `deleteTodoTask(todoTaskId, todoListId)`).
- Returns 204 on success.

**PATCH /api/google-tasks/items/:taskId/complete**
- Query: `?listId=<listId>`
- Calls `completeTodoTask(listId, taskId)`.
- Returns 200 `{ ok: true }` on success, 502 on failure.

**Add `useCompleteMutation`** (new export — gtasks-specific):
```ts
export function useCompleteMutation() {
  const { token } = useAuth();  // same pattern as other mutations
  return {
    mutate: async ({ item }: { item: CalendarItem }) => {
      if (item.source !== 'gtasks') return;  // Complete is gtasks-only
      const listId = item.sourceFields.listId as string;
      const res = await apiFetch(
        `/api/google-tasks/items/${item.id}/complete?listId=${encodeURIComponent(listId)}`,
        { method: 'PATCH' },
        token
      );
      if (!res.ok) {
        toast.error('Failed to mark complete');
        return;
      }
      // No undo for complete — per CONTEXT strict-hide rule.
      toast.success('Marked complete');
      // Optimistic: return the item.id so Calendar.tsx can add it to deletedIds
      return item.id;
    },
  };
}
```
